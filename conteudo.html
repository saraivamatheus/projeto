O que são APIs?

A grosso modo, códigos que simplificam processos complexos, APIs abstraem a complexidade de um processo. Javascript(JS), por exemplo, já é uma API por si só, pois, animações que seriam muito complexas de fazer por linguagens de baixo nível (C++, Assembly), são feitas de forma muito mais simples pela linguagem de alto nível: JS.

Analogia prática:

Para utilização de energia elétrica em sua casa, não é preciso pegar o cabeamento e ligar diretamente por fios um aparelho(além de ser ineficiente e difícil, é perigoso!), e sim, utilizar uma tomada para ligar o aparelho. Elucidando, a tomada é uma API de energia elétrica!

APIs da web

Fazem parte do seu navegador, sendo capazes de expor dados do navegador. Por exemplo, para manipulação de áudio, JS utiliza instruções simples para alterar o volume dela, mas, por trás dos panos, o navegador utiliza códigos complexos de baixo nível para processar. Abstração da complexidade!

APIs comuns de navegadores

APIs de manipulação de documentos - DOM(document object model) que permite manipular HTML e CSS - aplicando dinamicamente estilos a página, toda vez que aparece um pop-up, ou um novo conteúdo é mostrado, o DOM está em ação!

APIs que buscam dados no servidor

APIs para desenhar e manipular elementos gráficos
